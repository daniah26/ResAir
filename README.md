# ResAir
ResAir is an airline reservation system that  provides a better convenient and automated  administration and booking of airline tickets. With the assistance of the system multiple airlines  are able to effectively direct and manage a passengerâ€™s information, reserve and choose a seat  as well as handle the lifecycle said reservation.

# Name Decision
The name decision came from AIRline and REServation. It is short and provides meaning in english ( most spoken language ), therefor it is easy to remember.

## Purpose
The main purpose of ResAir is to create an automated airline reservation platform that allows passengers to search for flights, reserve seats, and manage bookings without manual intervention from airline staff.

## Objectives
1. Simplify the airline ticketing process through automation.
2. Allow multiple airlines to manage passenger reservations from one system.
3. Enable passengers to search, select, and book seats easily.
4. Maintain accurate passenger and flight data in a secure database.
5. Provide an admin panel for airlines to manage flight schedules and availability.
